# AdFlow

**Agent-powered ad marketplace for the open web.**

## Live Demo

**[https://eth-global-2026-adflow-git-dev-cytric.vercel.app/](https://eth-global-2026-adflow-git-dev-cytric.vercel.app/)**

Publishers list their sites, advertisers find them via AI agents, and payments stream on-chain via escrow contracts with delivery confirmed by a permissioned reporting layer.

## Project Structure

- `atomicagenticad/` — Main app (Next.js frontend + Foundry smart contracts)
- `shane-frontend-blog/` — Blog frontend
